SMS Shield — Yapay Zekâ ile SMS Koruması

SMS pumping / OTP fraud saldırılarını gerçek zamanlı tespit eden yapay zekâ destekli risk skorlama servisi. Kural motoru, makine öğrenmesi ve LLM analizi katmanlarıyla SMS maliyetlerinizi koruyun.

  • Python
  • FastAPI
  • Özel AI
  • Redis
  • PostgreSQL
  • React

Öne çıkanlar: Gerçek zamanlı risk skorlama, kural motoru, ML tespiti, LLM ikinci görüş, Shadow/Enforce modu, yönetim paneli.

Identity Resolution — Müşteri Kimlik Çözümleme

Telefon, ad, e-posta, adres ve cihaz verilerini probabilistik eşleştirmeyle birleştirerek aynı kişi/hane kümelerini saniyeler içinde çıkarır.

  • Python
  • Özel AI
  • FastAPI
  • PostgreSQL
  • React

Öne çıkanlar: Otomatik profil birleştirme, probabilistik eşleştirme, adres/hane analizi, küme yönetim paneli.
